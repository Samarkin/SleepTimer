# Sleep Timer

Mac OS X utility that can add the sleep timer functionality to the apps
that don't support it.

## What?

The application lives in the menubar, and can pause
[Movist](https://apps.apple.com/us/app/movist/id461788075) with a 5/30/60/90/120-minute delay.
The application also support cycling through the timer options
with a hotkey (Ctrl+Shift+Option+Command+T).

## How to build?

Open `SleepTimer.xcodeproj` in the latest version of Xcode.
